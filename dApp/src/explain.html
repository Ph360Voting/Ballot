Casting a vote

  • Click the button asking to input a Vote ID
  • Input the Ballot ID of the vote instance you wish to participate in.
  • Rank your choices.
  • Using MetaMask or a similar web3 interface, submit your vote.

Starting a new voting instance

  • Click the button asking to start a new ballot.
  • Choose your voting instance options, including what kind of voting instance you wish and who can particiapte in it.
  • Using MetaMask or a similar web3 interface, submit your voting instance to the Ethereum blockchain.
  • Store and share with the participants the ballet ID that will appear after the opening of the voting instance.
